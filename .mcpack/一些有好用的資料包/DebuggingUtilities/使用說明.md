# DebuggingUtilities (模組化日誌系統)

本資料包提供標準化、分級的除錯介面，利用原版 NBT 儲存區傳遞文字，讓你告別冗長且難以維護的 `/tellraw` 指令。

**支援遊戲版本:** `1.20.2+`

---

## 快速使用指南

在你的任何開發專案中，只需兩行指令即可印出帶有顏色前綴的格式化日誌：

# 1. 將欲輸出的文字或 JSON 文本寫入緩衝區
```mcfunction
data modify storage logger:buffer message set value '{"text":"這是一則測試訊息","color":"white"}'
```

# 2. 呼叫對應的日誌層級 (info / warn / error)
```mcfunction
function logger:info
```
