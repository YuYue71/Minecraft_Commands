# RandomNumbers (全域動態亂數引擎)

本資料包提供極度輕量, 基於 LCG 演算法的純計分板亂數生成工具, 可作為底層 API 供其他資料包直接調用.

**支援遊戲版本:** `1.20.2+`

---

## 快速使用指南

在你的任何資料包腳本或指令方塊中，依照以下三個步驟呼叫即可：


### 1. 設定你想要的亂數範圍 (例如: 15 到 80)
```mcfunction
scoreboard players set #min random_numbers 15
scoreboard players set #max random_numbers 80
```
### 2. 呼叫亂數引擎
```mcfunction
function random_numbers:core/generate
```
### 3. 讀取並使用結果 (執行後，#result 的值必定在 15~80 之間)
範例: 將算出的亂數轉移到你自己的專案變數中
```mcfunction
scoreboard players operation <你的自訂變數> <你的計分板> = #result random_numbers
```

> **開發備註：**
> 系統會在 `/reload` 或伺服器啟動時自動完成初始化與洗牌 (`#seed` 重置)，因此在日常調用時，**只需要給定 min 與 max，然後直接呼叫 generate 即可**，無需手動管理底層常數。
