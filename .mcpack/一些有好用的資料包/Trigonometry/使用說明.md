# Trigonometry (全域三角函數查表庫)

基於查表法 (Lookup Table) 打造的高效能三角函數計算核心. 所有運算結果均放大 **1000倍** 以適應計分板整數限制.

**支援遊戲版本:** `1.20.2+`

---

## 快速使用指南

在任何資料包中, 透過寫入 `#in_angle` 變數來呼叫計算:

### 1. 賦予角度值 (支援負數與大於 360 的數值, 系統會自動正規化)
```mcfunction
scoreboard players set #in_angle math 45
```
### 2. 呼叫計算函式
```mcfunction
function math:core/sin
function math:core/cos
```
### 3. 讀取結果 (此時 #out_sin 和 #out_cos 會得到 707)
### 範例: 將結果轉移至你的座標系變數中
```mcfunction
scoreboard players operation #particle_x example_vars = #out_cos math
scoreboard players operation #particle_z example_vars = #out_sin math
```

開發備註： > 由於運算結果放大了 1000 倍, 在套用到真實座標 (^ ^ ^) 或實體移動 (tp) 前, 請記得搭配微距傳送或利用除法 (/= 1000) 進行比例還原.
