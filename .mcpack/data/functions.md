# function/
## 注意事項
- function 資料夾內每個檔案必須以 `.mcfunction` 結尾
- 指令開頭不能有"/"
- 使用"#"可以寫註解
- 不能打指令跟註解以外的內容
- 一行一條指令
- 可以建立模組化結構管理 function
