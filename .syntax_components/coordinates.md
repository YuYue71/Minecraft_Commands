## 核心結構 (Core Structure)

```text
X Y Z

```

* Minecraft 世界採用三維笛卡爾座標系, 單位為格 (Blocks).

| 屬性 / 欄位 | 類型 | 預設 | 說明 |
| --- | --- | --- | --- |
| `X` | `float` | - | 東西向座標 (正值往東, 負值往西) |
| `Y` | `float` | - | 垂直高度座標 (正值往上, 負值往下) |
| `Z` | `float` | - | 南北向座標 (正值往南, 負值往北) |

## 屬性說明 (Attributes)

### `座標符號`

> 座標輸入的表示法.

| 值 / 範圍 | 說明 |
| --- | --- |
| `數字` | 絕對座標, 基於世界原點 (0, 0, 0) 的絕對位置 |
| `~` | 相對座標, 基於執行實體或方塊的當前位置 (如 `~ ~2 ~` 為上方 2 格) |
| `^` | 局部座標, 基於執行實體當前視線朝向 (如 `^ ^ ^5` 為面向前方 5 格) |

---

### `局部座標軸`

> 局部座標 `^` 的三個軸向定義.

| 值 | 說明 |
| --- | --- |
| `^第一個值` | 左右向 (正值往左, 負值往右) |
| `^第二個值` | 上下向 (正值往上, 負值往下) |
| `^第三個值` | 前後向 (正值往前, 負值往後) |
