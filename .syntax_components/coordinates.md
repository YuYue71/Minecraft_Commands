# 空間座標系統全指南（Coordinates）

## 三維空間軸向與方向基礎 (Minecraft Axis & Direction)
> Minecraft 採用右手笛卡爾座標系, 在地圖中定位任何方塊或實體的位置時, 必須遵循以下底層軸向定義

| 軸向變數 | 代表方向 | 數值增加（正數） | 數值減少（負數） | 遊戲內除錯畫面 (F3) 對照 |
| :--- | :--- | :--- | :--- | :--- |
| `X` | 東西向 (East / West) | 向東 (East) 移動 | 向西 (West) 移動 | Towards positive/negative X |
| `Y` | 垂直向 (Up / Down) | 向天空中 (Up) 升高 | 向地底深處 (Down) 降低 | 虛空邊界至天空高度限制區間 |
| `Z` | 南北向 (South / North) | 向南 (South) 移動 | 向北 (North) 移動 | Towards positive/negative Z |

## 三大座標表達式矩陣 (Coordinate Types)
> 在指令中輸入位置參數時, 共有三種核心表達方式, 用於應付不同的空間拓撲情境

| 座標型態 | 語法符號 | 基準點定義 (Origin) | 簡介說明與高階特徵 |
| :--- | :--- | :--- | :--- |
| **絕對座標** | `X Y Z` | 世界地圖的全域原點 `(0, 0, 0)` | 精準指向地圖上的固定幾何位置, 不隨執行者改變而偏移（如 `100 64 -200`） |
| **相對座標** | `~X ~Y ~Z` | 指令當前執行者 / 方塊的自身座標 | 以執行點為原點進行向量偏移（如 `~ ~1 ~` 代表執行者頭頂上方一格的空間） |
| **局部座標** | `^X ^Y ^Z` | 執行者當前的眼睛位置與頭部視線朝向 | 以視線為基準的局部座標（`^左/右 ^上/下 ^前/後`, 如 `^ ^ ^5` 代表準心正前方 5 格處） |
