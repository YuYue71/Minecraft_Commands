## 裝備與防具附魔 (Armor)

| ID | 說明 | 適用裝備 |
| --- | --- | --- |
| `protection` | 保護 (降低多數來源的傷害) | 盔甲全套 |
| `fire_protection` | 抗火性 (降低火焰傷害與燃燒時間) | 盔甲全套 |
| `feather_falling` | 輕盈 (降低墜落傷害) | 靴子 |
| `blast_protection` | 防爆 (降低爆炸傷害與擊退距離) | 盔甲全套 |
| `projectile_protection` | 防彈 (降低遠程拋射物傷害) | 盔甲全套 |
| `respiration` | 水下呼吸 (延長水下氧氣消耗時間) | 頭盔 |
| `aqua_affinity` | 親水性 (消除水下挖掘速度懲罰) | 頭盔 |
| `thorns` | 荊棘 (反彈部分近戰傷害給攻擊者) | 盔甲全套 |
| `depth_strider` | 深海漫遊 (提升水下移動速度) | 靴子 |
| `frost_walker` | 冰霜行者 (在水面上行走時結冰, 並免疫營火與岩漿塊傷害) | 靴子 |
| `soul_speed` | 靈魂疾行 (提升在靈魂沙與靈魂土壤上的移動速度) | 靴子 |
| `swift_sneak` | 迅捷潛行 (提升潛行時的移動速度) | 護腿 |

## 遠程武器附魔 (Ranged Weapons)

| ID | 說明 | 適用裝備 |
| --- | --- | --- |
| `power` | 力量 (增加弓的射擊傷害) | 弓 |
| `punch` | 衝擊 (增加弓的擊退距離) | 弓 |
| `flame` | 火矢 (使射出的箭矢著火, 可點燃 TNT) | 弓 |
| `infinity` | 無限 (射擊時不消耗普通箭矢) | 弓 |

## 工具與通用附魔 (Tools & General)

| ID | 說明 | 適用裝備 |
| --- | --- | --- |
| `efficiency` | 效率 (提升方塊挖掘與破壞速度) | 鎬, 斧, 鏟, 鋤, 剪刀 |
| `silk_touch` | 絲綢之觸 (挖掘方塊時掉落方塊本身而非其產物) | 鎬, 斧, 鏟, 鋤 |
| `unbreaking` | 耐久 (降低消耗裝備耐久度的機率) | 所有裝備 |
| `fortune` | 幸運 (增加方塊破壞時產物的掉落數量與機率) | 鎬, 斧, 鏟, 鋤 |
| `luck_of_the_sea` | 海之眷顧 (增加釣魚時獲得寶藏的機率, 降低垃圾機率) | 釣魚竿 |
| `lure` | 誘餌 (減少釣魚時等待魚咬鉤的秒數) | 釣魚竿 |
| `mending` | 修補 (吸收經驗球以自動恢復裝備耐久度) | 所有裝備 |
| `binding_curse` | 綁定詛咒 (裝備被穿上後無法在生存模式中卸下) | 盔甲全套, 南瓜, 鞘翅 |
| `vanishing_curse` | 消失詛咒 (玩家死亡時該裝備會直接消失而不掉落) | 所有裝備 |
