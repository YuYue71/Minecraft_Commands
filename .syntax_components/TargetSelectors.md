## 核心結構 (Core Structure)

```text
@selector[argument1=value,argument2=value]

```

* 用於動態篩選與指定遊戲內的一個或多個實體.

| 屬性 / 欄位 | 類型 | 預設 | 說明 |
| --- | --- | --- | --- |
| `選擇器類型` | `string` | - | 指定基本篩選目標類型 |
| `[篩選參數]` | `arguments` | - | 選填的過濾條件, 以逗號分隔 |

---

## 屬性說明 (Attributes)

### `選擇器類型`

> 基礎篩選目標定義.

| 值 | 說明 |
| --- | --- |
| `@p` | 距離執行地點最近的玩家 |
| `@r` | 隨機單一玩家 |
| `@a` | 所有存活的玩家 |
| `@e` | 所有存活的實體 (包含玩家與物件) |
| `@s` | 執行命令的當前實體 |

---

### `篩選參數`

> 用於精確過濾目標的引數清單. 支援 `!` 的參數代表可進行反向排除 (例如 `type=!player`).

| 值 / 範圍 | 說明 |
| --- | --- |
| `x`, `y`, `z` | 重新定義搜尋原點座標 (如 `x=10,y=20,z=30`) |
| `dx`, `dy`, `dz` | 定義搜尋體積的長寬高延伸量, 形成矩形邊界 |
| `distance=<範圍>` | 依距離篩選, 支援範圍標記 (如 `distance=..10` 為 10 格內) |
| `x_rotation=<範圍>` | 篩選垂直視角 (俯仰角), 範圍 -90 (上) 到 90 (下) |
| `y_rotation=<範圍>` | 篩選水平視角 (偏航角), 範圍 -180 到 180 |
| `level=<範圍>` | 篩選玩家的經驗等級 (如 `level=5..` 為大於等於 5 等) |
| `gamemode=<模式>` | 篩選遊戲模式 (如 `gamemode=survival`), 支援 `!` 排除 |
| `name=<名稱>` | 篩選指定名稱的實體, 支援 `!` 排除 |
| `type=<實體ID>` | 篩選指定實體類型 (如 `type=minecraft:zombie`), 支援 `!` 排除 |
| `team=<隊伍名>` | 篩選所屬隊伍, 支援 `!` 排除, 留空 (`team=`) 代表無隊伍 |
| `tag=<標籤>` | 篩選計分板標籤, 支援 `!` 排除, 留空 (`tag=`) 代表無任何標籤 |
| `limit=<整數>` | 限制最大回傳數量 |
| `sort=<排序法>` | 排序規則 (最近的`nearest`, 最遠的`furthest`, 隨機的`random`, 隨意的`arbitrary`) |
| `scores={<計分項>=<範圍>}` | 篩選符合特定計分板分數範圍的實體 |
| `advancements={<成就>=<bool>}` | 篩選特定進度/成就的完成狀態 |
| `predicate=<資源位置>` | 篩選符合指定資料包條件謂詞 (Predicate) 的實體, 支援 `!` 排除 |
