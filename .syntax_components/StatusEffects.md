## 正面效果 (Positive Effects)

| ID | 說明 |
| --- | --- |
| `speed` | 速度 (提升移動速度與視野廣度) |
| `haste` | 挖掘加速 (提升挖掘與攻擊速度) |
| `strength` | 力量 (提升近戰攻擊傷害) |
| `instant_health` | 瞬間治療 (立即恢復生命值, 對不死生物造成傷害) |
| `jump_boost` | 跳躍提升 (提升跳躍高度並降低掉落傷害) |
| `regeneration` | 恢復 (持續恢復生命值) |
| `resistance` | 抗性 (降低受到的所有傷害) |
| `fire_resistance` | 抗火性 (免疫火焰與岩漿傷害) |
| `water_breathing` | 水下呼吸 (在水下不會消耗氧氣並稍微增加視野亮度) |
| `invisibility` | 隱形 (使實體模型不可見, 降低怪物偵測距離) |
| `night_vision` | 夜視 (提升環境亮度至最大值, 包含水下視野) |
| `health_boost` | 生命提升 (增加最大生命值上限) |
| `absorption` | 傷害吸收 (提供額外的黃金心, 消耗後不自動恢復) |
| `saturation` | 飽和 (恢復飽和度與飢餓值) |
| `luck` | 幸運 (提升釣魚與戰利品表的高稀有度掉落率) |
| `slow_falling` | 緩降 (降低墜落速度並完全免疫掉落傷害) |
| `conduit_power` | 導管能量 (在水下時提供水下呼吸, 夜視與急迫效果) |
| `dolphins_grace` | 海豚的恩典 (大幅提升水下移動速度) |
| `hero_of_the_village` | 村莊英雄 (降低村民交易價格並獲得村民拋出的贈禮) |

## 負面效果 (Negative Effects)

| ID | 說明 |
| --- | --- |
| `slowness` | 緩速 (降低移動速度與視野廣度) |
| `mining_fatigue` | 挖掘疲勞 (大幅降低挖掘與攻擊速度) |
| `instant_damage` | 瞬間傷害 (立即造成魔法傷害, 對不死生物產生治療) |
| `nausea` | 反胃 (扭曲並晃動玩家視角) |
| `blindness` | 失明 (大幅限制視野距離, 無法奔跑或觸發物理暴擊) |
| `hunger` | 飢餓 (加速飢餓值消耗) |
| `weakness` | 虛弱 (降低近戰攻擊傷害) |
| `poison` | 中毒 (持續造成傷害, 但生命值最低只會降至半顆心, 不會致死) |
| `wither` | 凋零 (持續造成傷害, 與中毒不同, 凋零會致死) |
| `levitation` | 飄浮 (強制使實體不斷上升, 效果結束時可能承受掉落傷害) |
| `unluck` | 霉運 (降低釣魚與戰利品表的高稀有度掉落率) |
| `bad_omen` | 不祥之兆 (帶著此效果進入村莊範圍會觸發突襲) |
| `darkness` | 黑暗 (大幅降低視野並隨機閃爍變暗, 由伏守者或幽匿尖嘯體賦予) |

## 中性效果 (Neutral Effects)

| ID | 說明 |
