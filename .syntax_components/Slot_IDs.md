## 核心結構 (Core Structure)

```text
<slot_name>

```

* 用於 `/item` 等指令中指定要修改或替換的特定物品欄位.

| 屬性 / 欄位 | 類型 | 預設 | 說明 |
| --- | --- | --- | --- |
| `<slot_name>` | `string` | - | 指定實體或方塊容器的物品格索引名稱 |

## 欄位編號速查 (Slots)

### `裝備與武器欄位`

> 適用於玩家與多數可穿戴裝備之實體 (如殭屍, 骷髏, 盔甲座).

| 值 | 說明 |
| --- | --- |
| `weapon.mainhand` | 主手 (右手) 武器欄位 |
| `weapon.offhand` | 副手 (左手) 武器欄位 |
| `armor.head` | 頭部 (頭盔) 欄位 |
| `armor.chest` | 胸部 (胸甲, 鞘翅) 欄位 |
| `armor.legs` | 腿部 (護腿) 欄位 |
| `armor.feet` | 腳部 (靴子) 欄位 |

---

### `玩家專屬背包欄位`

> 僅限玩家實體 (`@a`, `@p`, `@r`, `@s` 若為玩家) 使用的專屬欄位空間.

| 值 / 範圍 | 說明 |
| --- | --- |
| `hotbar.0` .. `hotbar.8` | 快捷欄 (由左至右為 0 到 8) |
| `inventory.0` .. `inventory.26` | 玩家主要背包 (不含快捷欄與裝備欄), 索引由左上至右下排列 |
| `enderchest.0` .. `enderchest.26` | 玩家的終界箱背包 |

---

### `方塊與實體容器欄位`

> 適用於箱子, 熔爐, 漏斗等容器方塊, 以及儲物箱礦車, 漏斗礦車等實體容器.

| 值 / 範圍 | 說明 |
| --- | --- |
| `container.0` .. `container.53` | 泛用容器欄位. 單個箱子為 0~26, 大箱子為 0~53. 熔爐中 0 為輸入, 1 為燃料, 2 為輸出 |

---

### `特定生物專屬欄位`

> 針對具備特殊裝備格或自帶隱藏背包的生物實體.

| 值 / 範圍 | 說明 |
| --- | --- |
| `horse.saddle` | 鞍欄位 (適用於馬, 驢, 騾, 駱駝, 豬, 熾足獸) |
| `horse.armor` | 馬鎧欄位 (僅適用於馬) |
| `horse.chest` | 箱子裝備欄位 (適用於驢, 騾, 羊駝裝備箱子本身) |
| `horse.0` .. `horse.14` | 運輸動物的馱獸背包 (適用於已裝備箱子的驢, 騾, 羊駝) |
| `villager.0` .. `villager.7` | 村民的隱藏背包 (用於拾取與分享農作物, 食物) |
