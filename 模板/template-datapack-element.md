## JSON 結構

> **此元素只有單一結構時**，使用下方單一範本。
> **此元素依 `type` 有多種結構時**，刪除單一範本，改用多結構範本格式。

### 單一結構

```json
{
    "欄位A": "minecraft:值",
    "欄位B": {
        "子欄位": "值"
    },
    "選填欄位C": false
}
```

### 多結構（依 type 展開）

#### type = `"minecraft:值X"` 時

```json
{
    "type": "minecraft:值X",
    "欄位A": "值",
    "欄位B": 0
}
```

#### type = `"minecraft:值Y"` 時

```json
{
    "type": "minecraft:值Y",
    "欄位C": "值",
    "欄位D": []
}
```

## 欄位說明 (Fields)

### 根層欄位

| 欄位 | 類型 | 必填 | 預設 | 說明 |
|------|------|:----:|:----:|------|
| `欄位A` | `string` | ✓ | — | 說明此欄位的用途 |
| `欄位B` | `object` | ✓ | — | 說明此欄位的用途，子欄位見下方 |
| `選填欄位C` | `boolean` | | `false` | 說明此欄位的用途 |

### 子欄位：`欄位B`

| 欄位 | 類型 | 必填 | 預設 | 說明 |
|------|------|:----:|:----:|------|
| `子欄位` | `string` | ✓ | — | 說明 |

## 枚舉值速查

### `欄位A` 可填值

| 值 | 分類 | 說明 |
|----|------|------|
| `minecraft:值A` | 分類X | 說明 |
| `minecraft:值B` | 分類X | 說明 |
| `minecraft:值C` | 分類Y | 說明 `[1.XX+]` |
| ~~`minecraft:舊值`~~ | 分類Z | **Deprecated** — 說明，建議改用 `minecraft:新值` |

## 條件依賴 (Conditional Fields)

> 無欄位相依或互斥關係時，此區塊整段刪除

| 條件 | 生效欄位 | 失效 / 忽略欄位 |
|------|---------|---------------|
| `欄位A` = `"值A"` 時 | `欄位B` 必填 | `選填欄位C` 被忽略 |
| `欄位A` = `"值B"` 時 | `選填欄位C` 有效 | `欄位B` 可省略 |
