# 元素名稱 (`element_folder/`)

> **分類:** `資料包`　｜　**適用版本:** `JE 1.XX+`　｜　**路徑:** `data/<namespace>/element_folder/<名稱>.json`

- 引用格式：`namespace:名稱`（不含副檔名）

---

## 概覽 (Overview)

一句話說明此元素的功能與使用場景。

---

## JSON 結構

```json
{
    "必填欄位A": "<值>",
    "必填欄位B": {
        "子欄位": "<值>"
    },
    "選填欄位C": false
}
```

---

## 欄位說明 (Fields)

### 根層欄位

| 欄位 | 類型 | 必填 | 預設 | 說明 |
|------|------|:----:|:----:|------|
| `必填欄位A` | `string` | ✓ | — | 說明此欄位的用途 |
| `必填欄位B` | `object` | ✓ | — | 說明此欄位的用途，子欄位見下方 |
| `選填欄位C` | `boolean` | | `false` | 說明此欄位的用途 |

### `必填欄位B` 子欄位

| 欄位 | 類型 | 必填 | 預設 | 說明 |
|------|------|:----:|:----:|------|
| `子欄位` | `string` | ✓ | — | 說明 |

---

## 條件依賴（Conditional Fields）

> 僅在欄位之間存在相依或互斥關係時使用此區塊，否則刪除

| 條件 | 生效欄位 | 失效 / 忽略欄位 |
|------|---------|---------------|
| `必填欄位A` = `"值X"` 時 | `必填欄位B` 必填 | `選填欄位C` 被忽略 |
| `必填欄位A` = `"值Y"` 時 | `選填欄位C` 有效 | `必填欄位B` 可省略 |

---

## 枚舉值速查

### `必填欄位A` 可填值

| 值 | 分類 | 說明 |
|----|------|------|
| `"minecraft:值A"` | 分類X | 說明 |
| `"minecraft:值B"` | 分類X | 說明 |
| `"minecraft:值C"` | 分類Y | 說明 `[1.XX+]` |
