## 子分類A

> 一句話說明此子分類的功能與使用場景

**路徑：** `data/<namespace>/container_folder/子分類A/<名稱>.json`

### JSON 結構

```json
{
    "必填欄位A": "minecraft:值",
    "必填欄位B": {
        "子欄位": "值"
    },
    "選填欄位C": false
}
```

> **此子分類有多種 type 結構時**，每種 type 獨立一個結構範本，格式如下：

#### type = `"值X"` 時

```json
{
    "type": "minecraft:值X",
    "欄位A": "值",
    "欄位B": 0
}
```

#### type = `"值Y"` 時

```json
{
    "type": "minecraft:值Y",
    "欄位C": "值",
    "欄位D": []
}
```

### 欄位說明

#### 根層欄位

| 欄位 | 類型 | 必填 | 預設 | 說明 |
|------|------|:----:|:----:|------|
| `必填欄位A` | `string` | ✓ | — | 說明此欄位的用途 |
| `必填欄位B` | `object` | ✓ | — | 說明此欄位的用途，子欄位見下方 |
| `選填欄位C` | `boolean` | | `false` | 說明此欄位的用途 |

#### 子欄位：`必填欄位B`

| 欄位 | 類型 | 必填 | 預設 | 說明 |
|------|------|:----:|:----:|------|
| `子欄位` | `string` | ✓ | — | 說明 |

### 枚舉值速查

#### `必填欄位A` 可填值

| 值 | 分類 | 說明 |
|----|------|------|
| `minecraft:值A` | 分類X | 說明 |
| `minecraft:值B` | 分類X | 說明 |
| `minecraft:值C` | 分類Y | 說明 `[1.XX+]` |
| ~~`minecraft:舊值`~~ | 分類Z | **Deprecated** — 說明，建議改用 `minecraft:新值` |

### 條件依賴

> 無欄位相依或互斥關係時，此區塊整段刪除

| 條件 | 生效欄位 | 失效 / 忽略欄位 |
|------|---------|---------------|
| `必填欄位A` = `"值A"` 時 | `必填欄位B` 必填 | `選填欄位C` 被忽略 |
