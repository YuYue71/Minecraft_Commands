# `/kill`（擊殺實體）

* > 立即移除指定實體
* > 可用於清除玩家或生物

## 語法結構 (Syntax)

```commands id="kill"
/kill <目標>
```

## 參數與引數拆解 (Arguments)

| 參數名稱   | 功能與語義說明 |
| ------ | ------- |
| `<目標>` | 要被移除的實體 |

---

## 參數枚舉列表 (Parameter Enumeration)

### 目標

| 參數        | 說明         |
| --------- | ---------- |
| `<目標選擇器>` | 可選擇玩家或任何實體 |
| `@s`      | 自身         |
| `@p`      | 最近玩家       |
| `@a`      | 全部玩家       |
| `@e`      | 所有實體       |
| `@r`      | 隨機玩家       |
