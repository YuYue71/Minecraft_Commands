# `/debug`（除錯指令）

* > 用於分析伺服器效能與執行狀態
* > 可開始或停止效能分析
* > 可產生效能分析報告

## 語法結構 (Syntax)

```commands
/debug <操作>
```

---

## 參數與引數拆解 (Arguments)

> 詳細解構語法中出現的每一個變數之填寫規範與底層資料型態

| 參數名稱          | 功能與語義說明 |
| ------------- | ------- |
| `[必填]` `<操作>` | 指定除錯模式  |

---

## 參數枚舉列表 (Parameter Enumeration)

### 操作

| 參數           | 說明          |
| ------------ | ----------- |
| `<start>`    | 開始效能分析      |
| `<stop>`     | 停止效能分析並輸出報告 |
| `<function>` | 執行函式效能分析    |
|              |             |
