# `/experience`（經驗值指令）

* > 用於查詢或修改玩家經驗值
* > 可增減等級或經驗值數量
* > 影響附魔與升級系統

---

## 語法結構 (Syntax)

```commands id="experience0"
/experience <操作> <目標> <數值> <類型>
```

---

## 參數與引數拆解 (Arguments)

> 詳細解構語法中出現的每一個變數之填寫規範與底層資料型態

| 參數名稱          | 功能與語義說明     |
| ------------- | ----------- |
| `[必填]` `<操作>` | 指定經驗操作類型    |
| `[必填]` `<目標>` | 指定玩家或目標選擇器  |
| `[必填]` `<數值>` | 設定增加或減少的經驗量 |
| `[選填]` `<類型>` | 指定以等級或點數計算  |

---

## 參數枚舉列表 (Parameter Enumeration)

### 操作

| 參數      | 說明    |
| ------- | ----- |
| `add`   | 增加經驗  |
| `set`   | 設定經驗值 |
| `query` | 查詢經驗值 |

---

### 類型

| 參數       | 說明       |
| -------- | -------- |
| `levels` | 以等級為單位   |
| `points` | 以經驗點數為單位 |

---

### 數值

| 參數     | 說明                                        |
| ------ | ----------------------------------------- |
| `<數值>` | 最小值：-2147483648 / 最大值：2147483647 / 支援負數：是 |
