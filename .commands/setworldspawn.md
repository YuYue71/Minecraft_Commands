# `/setworldspawn`（世界重生點設定指令）

* > 用於設定世界的全域重生點
* > 玩家未設定個人重生點時將在此出生
* > 可選擇指定座標位置

## 語法結構 (Syntax)

```commands id="setworldspawn0"
/setworldspawn <座標>
```

---

## 參數與引數拆解 (Arguments)

> 詳細解構語法中出現的每一個變數之填寫規範與底層資料型態

| 參數名稱          | 功能與語義說明   |
| ------------- | --------- |
| `[選填]` `<座標>` | 指定世界重生點位置 |
