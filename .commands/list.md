# `/list`（玩家列表指令）

* > 用於顯示目前伺服器在線玩家清單
* > 可顯示玩家數量與上限
* > 可選擇性顯示具備特定標籤或過濾條件的玩家資訊

## 語法結構 (Syntax)

```commands id="p0l3x7"
/list <過濾>
```

---

## 參數與引數拆解 (Arguments)

> 詳細解構語法中出現的每一個變數之填寫規範與底層資料型態

| 參數名稱          | 功能與語義說明     |
| ------------- | ----------- |
| `[選填]` `<過濾>` | 指定顯示條件或篩選模式 |

---

## 參數枚舉列表 (Parameter Enumeration)

### 過濾

| 參數      | 說明           |
| ------- | ------------ |
| `uuids` | 顯示玩家 UUID 清單 |
|         |              |
