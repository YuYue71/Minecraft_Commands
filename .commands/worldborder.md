# `/worldborder`（世界邊界指令）

* > 用於設定或查詢世界邊界範圍
* > 控制玩家可活動的最大區域
* > 可動態縮放或固定邊界

## 語法結構 (Syntax)

```commands id="worldborder0"
/worldborder <操作> <值>
```

---

## 參數與引數拆解 (Arguments)

> 詳細解構語法中出現的每一個變數之填寫規範與底層資料型態

| 參數名稱          | 功能與語義說明     |
| ------------- | ----------- |
| `[必填]` `<操作>` | 指定世界邊界控制模式  |
| `[選填]` `<值>`  | 設定對應數值或目標位置 |

---

## 參數枚舉列表 (Parameter Enumeration)

### 操作

| 參數        | 說明        |
| --------- | --------- |
| `set`     | 設定世界邊界大小  |
| `add`     | 增加或減少世界邊界 |
| `center`  | 設定世界邊界中心  |
| `damage`  | 設定超出邊界傷害  |
| `warning` | 設定警告距離與時間 |
| `get`     | 查詢世界邊界大小  |

---

### 值（set / add）

| 參數     | 說明          |
| ------ | ----------- |
| `<數值>` | 邊界直徑大小（方塊數） |
