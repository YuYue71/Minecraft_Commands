# `/pardon-ip`（解除IP封鎖指令）

* > 用於解除指定IP位址的封鎖狀態
* > 解除後該IP可重新連線伺服器
* > 僅限具有足夠權限的管理員使用

## 語法結構 (Syntax)

```commands id="h2p8qv"
/pardon-ip <IP位址>
```

---

## 參數與引數拆解 (Arguments)

> 詳細解構語法中出現的每一個變數之填寫規範與底層資料型態

| 參數名稱            | 功能與語義說明      |
| --------------- | ------------ |
| `[必填]` `<IP位址>` | 指定要解除封鎖的IP地址 |
