# `/whitelist`（白名單管理指令）

* > 用於管理伺服器白名單
* > 控制允許加入伺服器的玩家
* > 可啟用, 停用與新增或移除玩家

## 語法結構 (Syntax)

```commands id="l2v9qp"
/whitelist <操作> <玩家>
```

---

## 參數與引數拆解 (Arguments)

> 詳細解構語法中出現的每一個變數之填寫規範與底層資料型態

| 參數名稱          | 功能與語義說明   |
| ------------- | --------- |
| `[必填]` `<操作>` | 指定白名單管理動作 |
| `[選填]` `<玩家>` | 指定玩家名稱    |

---

## 參數枚舉列表 (Parameter Enumeration)

### 操作

| 參數       | 說明       |
| -------- | -------- |
| `on`     | 啟用白名單    |
| `off`    | 關閉白名單    |
| `add`    | 加入玩家至白名單 |
| `remove` | 移除玩家白名單  |
| `list`   | 顯示白名單玩家  |
| `reload` | 重新載入白名單  |
