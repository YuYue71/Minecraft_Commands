# `/datapack`（資料包管理指令）

* > 用於管理世界中已載入的資料包
* > 可啟用, 停用與查詢資料包

## 語法結構 (Syntax)

```commands
/datapack <操作> <名稱> <參數>
```

---

## 參數與引數拆解 (Arguments)

> 詳細解構語法中出現的每一個變數之填寫規範與底層資料型態

| 參數名稱          | 功能與語義說明   |
| ------------- | --------- |
| `[必填]` `<操作>` | 指定資料包操作模式 |
| `[選填]` `<名稱>` | 資料包名稱     |
| `[選填]` `<參數>` | 操作所需附加參數  |

---

## 參數枚舉列表 (Parameter Enumeration)

### 操作

| 參數        | 說明    |
| --------- | ----- |
| `disable` | 停用資料包 |
| `enable`  | 啟用資料包 |
| `list`    | 列出資料包 |

---

### 參數

| 參數          | 說明         |
| ----------- | ---------- |
| `after`     | 載入於指定資料包之後 |
| `before`    | 載入於指定資料包之前 |
| `first`     | 移至最前方      |
| `last`      | 移至最後方      |
| `available` | 顯示可用資料包    |
| `enabled`   | 顯示已啟用資料包   |
