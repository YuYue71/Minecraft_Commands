# `/op`（授予管理員權限指令）

* > 用於將玩家提升為伺服器管理員
* > 被授權玩家可使用高權限指令
* > 僅限具有伺服器控制權的使用者執行

## 語法結構 (Syntax)

```commands id="k9x2qp"
/op <玩家>
```

---

## 參數與引數拆解 (Arguments)

> 詳細解構語法中出現的每一個變數之填寫規範與底層資料型態

| 參數名稱          | 功能與語義說明         |
| ------------- | --------------- |
| `[必填]` `<玩家>` | 指定要授予管理員權限的玩家名稱 |
