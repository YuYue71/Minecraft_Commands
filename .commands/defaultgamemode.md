# `/defaultgamemode`（預設遊戲模式指令）

* > 用於設定世界的預設遊戲模式
* > 影響新加入玩家的遊戲模式
* > 不會直接改變目前在線玩家的遊戲模式

## 語法結構 (Syntax)

```commands
/defaultgamemode <遊戲模式>
```

---

## 參數與引數拆解 (Arguments)

> 詳細解構語法中出現的每一個變數之填寫規範與底層資料型態

| 參數名稱            | 功能與語義說明     |
| --------------- | ----------- |
| `[必填]` `<遊戲模式>` | 指定世界預設的遊戲模式 |

---

## 參數枚舉列表 (Parameter Enumeration)

### 遊戲模式

| 參數            | 說明   |
| ------------- | ---- |
| `<survival>`  | 生存模式 |
| `<creative>`  | 創造模式 |
| `<adventure>` | 冒險模式 |
| `<spectator>` | 旁觀模式 |
