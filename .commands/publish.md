# `/publish`（區域網開放指令）

* > 用於將單人世界開放至區域網連線
* > 允許其他玩家透過區域網加入世界
* > 可選擇指定連線埠設定

## 語法結構 (Syntax)

```commands id="x8m1qp"
/publish <設定>
```

---

## 參數與引數拆解 (Arguments)

> 詳細解構語法中出現的每一個變數之填寫規範與底層資料型態

| 參數名稱          | 功能與語義說明      |
| ------------- | ------------ |
| `[選填]` `<設定>` | 指定區域網開放的連線設定 |

---

## 參數枚舉列表 (Parameter Enumeration)

### 設定

| 參數       | 說明       |
| -------- | -------- |
| `<port>` | 指定開放的連線埠 |
| `<auto>` | 自動分配連線設定 |
