# `/time`（時間控制指令）

* > 用於查詢或設定世界時間
* > 可控制日夜循環與時間刻度
* > 影響全世界時間狀態

## 語法結構 (Syntax)

```commands id="time0"
/time <操作> <值>
```

---

## 參數與引數拆解 (Arguments)

> 詳細解構語法中出現的每一個變數之填寫規範與底層資料型態

| 參數名稱          | 功能與語義說明     |
| ------------- | ----------- |
| `[必填]` `<操作>` | 指定時間控制模式    |
| `[選填]` `<值>`  | 設定時間數值或查詢條件 |

---

## 參數枚舉列表 (Parameter Enumeration)

### 操作

| 參數      | 說明           |
| ------- | ------------ |
| `set`   | 設定世界時間       |
| `add`   | 在當前時間基礎上增加刻數 |
| `query` | 查詢當前時間       |

---

### 值（set 模式）

| 參數           | 說明        |
| ------------ | --------- |
| `<day>`      | 設為白天      |
| `<noon>`     | 設為正午      |
| `<night>`    | 設為夜晚      |
| `<midnight>` | 設為午夜      |
| `<整數>`       | 指定 tick 數 |

---

### 值（query 模式）

| 參數         | 說明        |
| ---------- | --------- |
| `daytime`  | 查詢當前一天內時間 |
| `gametime` | 查詢世界總運行時間 |
