# `/forceload`（強制區塊載入）

* > 強制指定區塊持續載入
* > 即使玩家遠離也不會卸載

## 語法結構 (Syntax)

```commands id="forceload"
/forceload add <區塊座標>
/forceload add <起始區塊座標> <結束區塊座標>
/forceload remove <區塊座標>
/forceload remove <起始區塊座標> <結束區塊座標>
/forceload query
```

## 參數與引數拆解 (Arguments)

| 參數名稱       | 功能與語義說明 |
| ---------- | ------- |
| `<動作>`     | 指定操作類型  |
| `<區塊座標>`   | 單一區塊座標  |
| `<起始區塊座標>` | 區塊範圍起點  |
| `<結束區塊座標>` | 區塊範圍終點  |

---

## 參數枚舉列表 (Parameter Enumeration)

### 動作

| 參數       | 說明         |
| -------- | ---------- |
| `add`    | 新增強制載入區塊   |
| `remove` | 移除強制載入區塊   |
| `query`  | 查詢當前強制載入區塊 |

---

## 數值規則

### 區塊座標

| 參數     | 說明                                |
| ------ | --------------------------------- |
| `<數值>` | 最小值 -1875000 最大值 1875000 是否支援負數 是 |
