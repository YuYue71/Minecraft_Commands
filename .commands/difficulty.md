# `/difficulty`（難度設定指令）

* > 用於設定世界遊戲難度
* > 影響生存壓力與怪物生成強度
* > 可即時生效於伺服器或單人世界

## 語法結構 (Syntax)

```commands id="k8p1xd"
/difficulty <難度>
```

---

## 參數與引數拆解 (Arguments)

> 詳細解構語法中出現的每一個變數之填寫規範與底層資料型態

| 參數名稱          | 功能與語義說明    |
| ------------- | ---------- |
| `[必填]` `<難度>` | 設定世界遊戲難度等級 |

---

## 參數枚舉列表 (Parameter Enumeration)

### 難度

| 參數         | 說明   |
| ---------- | ---- |
| `peaceful` | 和平模式 |
| `easy`     | 簡單模式 |
| `normal`   | 普通模式 |
| `hard`     | 困難模式 |
