# `/spreadplayers`（隨機分散）

* > 將目標實體隨機分散到指定區域
* > 可控制距離與安全半徑

## 語法結構 (Syntax)

```commands id="spreadplayers"
/spreadplayers <中心X> <中心Z> <最小間距> <最大範圍> <是否保持原地> <目標>
```

## 參數與引數拆解 (Arguments)

| 參數名稱       | 功能與語義說明   |
| ---------- | --------- |
| `<中心X>`    | 分散區域中心X座標 |
| `<中心Z>`    | 分散區域中心Z座標 |
| `<最小間距>`   | 實體之間最小距離  |
| `<最大範圍>`   | 隨機分散半徑範圍  |
| `<是否保持原地>` | 是否保留Y軸高度  |
| `<目標>`     | 被分散的實體    |

---

## 參數枚舉列表 (Parameter Enumeration)

### 是否保持原地

| 參數      | 說明       |
| ------- | -------- |
| `true`  | 保持原Y軸高度  |
| `false` | 重新計算地表高度 |

---

## 數值規則

### 中心座標

| 參數     | 說明                                  |
| ------ | ----------------------------------- |
| `<數值>` | 最小值 -30000000 最大值 30000000 是否支援負數 是 |

### 最小間距

| 參數     | 說明                            |
| ------ | ----------------------------- |
| `<數值>` | 最小值 0 最大值 2147483647 是否支援負數 否 |

### 最大範圍

| 參數     | 說明                          |
| ------ | --------------------------- |
| `<數值>` | 最小值 0 最大值 30000000 是否支援負數 否 |
