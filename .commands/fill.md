# `/fill`（方塊填充）

* > 在指定區域內填充方塊
* > 可控制替換模式與填充方式

## 語法結構 (Syntax)

```commands id="fill"
/fill <起始座標> <結束座標> <方塊ID> <填充模式>
```

## 參數與引數拆解 (Arguments)

| 參數名稱     | 功能與語義說明 |
| -------- | ------- |
| `<起始座標>` | 區域第一個角落 |
| `<結束座標>` | 區域第二個角落 |
| `<方塊ID>` | 要填充的方塊  |
| `<填充模式>` | 填充行為模式  |

## 參數枚舉列表 (Parameter Enumeration)

### 填充模式

| 參數        | 說明       |
| --------- | -------- |
| `replace` | 直接替換所有方塊 |
| `destroy` | 破壞原方塊後填充 |
| `keep`    | 只填充空氣位置  |
| `outline` | 只填充外框    |
| `hollow`  | 中空填充     |

---

## 數值規則

### 座標範圍

| 參數     | 說明                                  |
| ------ | ----------------------------------- |
| `<數值>` | 最小值 -30000000 最大值 30000000 是否支援負數 是 |
