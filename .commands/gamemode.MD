# `/gamemode`（遊戲模式指令）

* > 用於切換玩家遊戲模式
* > 影響生存, 創造, 冒險與旁觀狀態
* > 可直接作用於單一或多個玩家

---

## 語法結構 (Syntax)

```commands id="gamemode0"
/gamemode <模式> <目標>
```

---

## 參數與引數拆解 (Arguments)

> 詳細解構語法中出現的每一個變數之填寫規範與底層資料型態

| 參數名稱          | 功能與語義說明    |
| ------------- | ---------- |
| `[必填]` `<模式>` | 指定玩家遊戲模式   |
| `[選填]` `<目標>` | 指定目標玩家或選擇器 |

---

## 參數枚舉列表 (Parameter Enumeration)

### 模式

| 參數          | 說明   |
| ----------- | ---- |
| `survival`  | 生存模式 |
| `creative`  | 創造模式 |
| `adventure` | 冒險模式 |
| `spectator` | 旁觀模式 |
