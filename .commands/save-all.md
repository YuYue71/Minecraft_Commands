# `/save-all`（世界儲存指令）

* > 用於強制將伺服器資料寫入磁碟
* > 可確保當前世界狀態被完整保存
* > 可選擇是否暫時關閉自動儲存

## 語法結構 (Syntax)

```commands id="a9d4kv"
/save-all <模式>
```

---

## 參數與引數拆解 (Arguments)

> 詳細解構語法中出現的每一個變數之填寫規範與底層資料型態

| 參數名稱          | 功能與語義說明  |
| ------------- | -------- |
| `[選填]` `<模式>` | 指定儲存執行方式 |

---

## 參數枚舉列表 (Parameter Enumeration)

### 模式

| 參數      | 說明       |
| ------- | -------- |
| `flush` | 強制立即寫入磁碟 |
