# `/weather`（天氣指令）

* > 用於查詢或設定天氣狀態
* > 可改變世界天氣效果
* > 影響降雨, 雷雨, 晴天等環境狀態

## 語法結構 (Syntax)

```commands id="weather0"
/weather <天氣類型> <時間>
```

---

## 參數與引數拆解 (Arguments)

> 詳細解構語法中出現的每一個變數之填寫規範與底層資料型態

| 參數名稱            | 功能與語義說明  |
| --------------- | -------- |
| `[必填]` `<天氣類型>` | 設定世界天氣狀態 |
| `[選填]` `<時間>`   | 指定天氣持續時間 |

---

## 參數枚舉列表 (Parameter Enumeration)

### 天氣類型

| 參數        | 說明 |
| --------- | -- |
| `clear`   | 晴天 |
| `rain`    | 下雨 |
| `thunder` | 雷雨 |

---

### 時間

| 參數     | 說明         |
| ------ | ---------- |
| `<整數>` | 持續 tick 時間 |
