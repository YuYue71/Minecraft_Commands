# `/setblock`（方塊設置）

* > 在指定座標放置或替換單一方塊
* > 可控制替換規則與破壞行為

## 語法結構 (Syntax)

```commands id="setblock"
/setblock <座標> <方塊ID> <替換模式>
```

## 參數與引數拆解 (Arguments)

| 參數名稱     | 功能與語義說明 |
| -------- | ------- |
| `<座標>`   | 放置方塊的位置 |
| `<方塊ID>` | 要放置的方塊  |
| `<替換模式>` | 是否覆蓋原方塊 |

---

## 參數枚舉列表 (Parameter Enumeration)

### 替換模式

| 參數        | 說明       |
| --------- | -------- |
| `replace` | 直接覆蓋原方塊  |
| `destroy` | 破壞原方塊後放置 |
| `keep`    | 只在空氣位置放置 |
