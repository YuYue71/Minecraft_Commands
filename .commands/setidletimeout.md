# `/setidletimeout`（閒置踢出時間設定指令）

* > 用於設定玩家閒置一段時間後自動踢出
* > 單位為分鐘
* > 僅影響伺服器設定，不影響已登入玩家即時狀態

## 語法結構 (Syntax)

```commands id="c1p7xz"
/setidletimeout <時間>
```

---

## 參數與引數拆解 (Arguments)

> 詳細解構語法中出現的每一個變數之填寫規範與底層資料型態

| 參數名稱          | 功能與語義說明           |
| ------------- | ----------------- |
| `[必填]` `<時間>` | 設定玩家閒置踢出時間, 單位為分鐘 |

---

## 參數枚舉列表 (Parameter Enumeration)

### 時間

| 參數     | 說明       |
| ------ | -------- |
| `<0>`  | 關閉閒置踢出機制 |
| `<整數>` | 設定閒置分鐘數  |
