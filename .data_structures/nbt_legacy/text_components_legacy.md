# 數據標籤：`Raw JSON 文本 NBT 格式` (Legacy Text Component)

## 基礎元數據 (Metadata)
> 用於快速查閱該數據結構的適用版本與容器型態

| 元數據項目 | 數值 / 狀態 | 備註說明 |
| :--- | :--- | :--- |
| **適用版本** | 1.20.4 以前所有舊版本 (預設相容) | 1.20.5+ 以上新版環境將無法直接以此格式解碼 |
| **底層資料型態**| NBT String (字串化 JSON 物件) | 舊版必須在 `Name` 或 `Lore` 標籤內以單引號將整個 JSON 包裹成「字串」 |
| **適用對象** | 物品 NBT、`/tellraw`、`/title` | 用於所有舊版涉及「富文本（Rich Text）」顯示的語法元件 |

---

## 語法結構 (Syntax)
> 在 1.20.4 以前, 物品的自訂名稱必須嚴格嵌套在 `display` 標籤下的 `Name` 字串內, 直接輸出文本則無需

###### 名稱賦予
```json
{display:{Name:'{"text":"<文字內容>","color":"<顏色>","bold":<布林值>}'}}
```
###### 文字輸出
```json
{"text":"<文字內容>","color":"<顏色>","bold":<布林值>}
```
###### 字串拼接
```json
'[{"text":"<文字內容>", "color":"<顏色>"}, {"text":"<文字內容>", "color":"<顏色>"}]'
```

## 參數與引數拆解 (Arguments)
> 詳細解構語法中出現的每一個變數之填寫規範與底層資料型態

| 參數名稱 | 接受資料型態 | 功能與語義說明 |
| --- | --- | --- |
| `[必填]` `"text"` | String | 具體要顯示在畫面上的純文字內容 |
| `[選填]` `"color"` | String | 文字的顏色（支援 `red`, `gold` 等內建字串, 或 16 進位 Hex 色碼如 `#FF5555`） |
| `[選填]` `"bold"` | Boolean | 是否將字體加粗變更為重音格式（`true` / `false`） |
| `[選填]` `"italic"` | Boolean | 是否將字體變更為斜體格式（注意：物品自訂名稱在舊版預設為斜體, 填 `false` 可轉正） |
| `[選填]` `"underlined"` | Boolean | 是否在文字下方追加一條底線 |
| `[選填]` `"strikethrough"` | Boolean | 是否在文字中央字元處追加一條刪除線 |
| `[選填]` `"obfuscated"` | Boolean | 是否開啟「動態亂碼」效果（舊版樣式代碼的 `§k`） |
